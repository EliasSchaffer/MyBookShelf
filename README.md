# MyBookShelf

MyBookShelf is an open-source Java application designed to help users organize and track their personal book collection. Whether you're an avid reader or a casual book lover, MyBookShelf allows you to manage your reading list, track books you've read, and plan future reads all in one place.

## Features

- **Book Collection Management**: Add, edit, and remove books from your collection.
- **Customizable Categories**: Organize your books by genres, authors, or custom tags.
- **Bookshelf Overview**: View all your books in an organized and easy-to-use interface.
- **Search Functionality**: Quickly search for books by title, author, or category.
- **Reading Goals**: Set and track your reading goals throughout the year.

## Installation

### Prerequisites

Ensure you have the following installed:

- **Java Development Kit (JDK)** version 8 or higher
- **Gradle** (for building and managing dependencies)

### Steps to Install

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/yourusername/MyBookShelf.git
   ```
   
2. Navigate to the project directory:

   ```bash
   cd MyBookShelf
   ```

3. Build the project with Gradle:

   ```bash
    ./gradlew build
   ```

4. Run the Application

   ```bash
    ./gradlew run
   ```

void com.example.mybookshelf.BookRecommendationFlow.addChatMessage ( String text,
String senderType )
inline

Adds a chat message to the UI based on the provided text and sender type. The method creates a TextView for the message, sets its properties such as color, background, and layout, and adds it to the GridLayout in the main activity. It also ensures that the ScrollView scrolls to show the latest message.

Parameters
textThe text content of the chat message.
senderTypeThe type of the sender, either "user" or another identifier for system messages.

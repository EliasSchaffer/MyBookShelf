static void com.example.mybookshelf.apis.AiAPI.fetchResponse ( final String question,
final ApiResponseCallback callback )
inlinestatic

Fetches a response from an AI service based on a given question.

This method constructs a JSON payload with system and user messages, sends it to an AI service, and handles the response or failure through callbacks. It logs the request payload and processes the response body if successful, passing it to the success callback. In case of errors, it passes an appropriate error message to the failure callback.

Parameters
questionThe user's question to be sent to the AI service.
callbackThe callback interface to handle the AI service response or failure.

Handles the response from a network call, passing it to the appropriate callback method.

Handles request failure by notifying the callback and printing the stack trace.

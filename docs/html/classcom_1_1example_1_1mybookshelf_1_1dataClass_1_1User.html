◆ removeBook()

void com.example.mybookshelf.dataClass.User.removeBook ( Book book,
Context main,
RecyclerView container )
inline

Removes a book from both the local list and the user interface.

Iterates through the local book list to find and remove the specified book. Updates the UI by removing the corresponding visual representation of the book. If the book is marked as being in the database, attempts to remove it from there as well. Handles exceptions that may occur during database operations and provides user feedback.

Parameters
bookthe Book object representing the book to be removed
mainthe Context object for UI operations
containerthe RecyclerView container where the book's visual part is located

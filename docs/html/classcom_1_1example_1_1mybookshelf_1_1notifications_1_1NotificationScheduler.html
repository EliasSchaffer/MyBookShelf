◆ scheduleOneTimeNotification()

static void com.example.mybookshelf.notifications.NotificationScheduler.scheduleOneTimeNotification ( Context context,
LocalDateTime dateTime,
int requestCode )
inlinestatic

Schedules a one-time notification at a specified date and time.

This method sets up an alarm to trigger a notification at the given LocalDateTime. It checks if the provided date and time is in the past, logs a warning, and returns early if true. Otherwise, it creates an Intent with necessary extras and uses PendingIntent to schedule the notification using AlarmManager. The method also handles permission checks for scheduling exact alarms on newer Android versions.

void com.example.mybookshelf.LayoutManager.CostumeListAdapter.bindBookData ( BookViewHolder holder,
Book book,
int position )
inlineprivate

Binds data from a Book object to a BookViewHolder.

This method updates the UI components of a book item in the RecyclerView, including loading the book's image, setting its details, handling notes, and managing click events for removal and navigation. It also ensures that any existing text watchers are removed before adding new ones to prevent duplicate callbacks. Additionally, it manages asynchronous database operations for retrieving and updating notes.

Parameters
holderThe BookViewHolder instance holding the book's UI components.
bookThe Book object containing the data to bind.
positionThe position of the book in the RecyclerView.

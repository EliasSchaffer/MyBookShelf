This method iterates through a list of BarEntry objects and checks each entry's year-month value. It ensures that the year is between 2000 and 2030 and the month is within the range of 1 to 12. Only entries meeting these conditions are added to the validEntries list, which is then returned.

◆ showRegister()

void com.example.mybookshelf.UIMaster.showRegister ( )
inline

Displays the registration screen and sets up event listeners for user interactions.

This method switches the main activity's content view to the registration layout, initializes UI components, and registers click listeners for the register button and the switch-to-login button. When the register button is clicked, it calls MainActivity#handleRegister(EditText, EditText, EditText, EditText) to process user input and register a new account. If an exception occurs during this process, it throws a runtime exception. Similarly, when the switch-to-login button is clicked, it attempts to show the login screen by calling showLogin(), handling exceptions in the same manner.
